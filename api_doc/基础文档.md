## 环境名称
|   | 环境名称  | 全称（团队使用）    | 简称（个人使用）    |
|---|-------|-------------|-------------|
| * | 本地环境  | local       |             |
| * | 开发环境  | development | dev         |
| * | 测试环境  | testing     | test        |
|   | 预生产环境 | staging     | stag        |
| * | 开发环境  | production  | pro/product |

> 个人使用可使用简称，团队使用一律全称。包括环境配置的时候，要用全称
